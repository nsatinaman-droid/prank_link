# Fake Hacker Screen Prank Web App

## Description
This project is a fun and harmless "Fake Hacker Screen" web app designed to simulate a hacking experience. It features animated text, a progress bar, and randomly generated fake data to create an entertaining prank for friends.

## Project Structure
```
link_prank
├── index.html
├── styles.css
├── script.js
└── README.md
```

## Files Overview

- **index.html**: The main HTML document that structures the web app. It includes all necessary elements for the hacker screen, such as a full-screen container, animated text display, a progress bar, and a button to share the prank. It links to the CSS and JavaScript files.

- **styles.css**: Contains the CSS styles for the web app. It sets a full-screen dark theme with a black background, green text in a monospace font, and styles for the animated text and progress bar. The design is responsive and centered for mobile screens, preventing scrolling.

- **script.js**: This file includes the JavaScript code that handles the animated text display, simulates the hacking process, generates random fake data (IP address, location, device type), and controls the progress bar. It also includes the final prank message and the functionality for the share button.

## How to Run the App Locally
1. Clone the repository or download the project files.
2. Open the `index.html` file in your web browser.
3. Enjoy the prank!

## Deployment
You can easily deploy this app using GitHub Pages or any static hosting service. Simply upload the project files to your hosting service and access the `index.html` file.
